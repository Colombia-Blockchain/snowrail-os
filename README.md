# SnowRail Landing Page

Landing page for SnowRail - an Agentic Treasury Orchestrator on Cronos that unifies banks, stablecoins, and DeFi into a programmable liquidity layer.

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS v4
- lucide-react

## Getting Started

```bash
pnpm install
pnpm dev
```

## Scripts

| Command        | Description                      |
| -------------- | -------------------------------- |
| `pnpm dev`     | Start dev server with HMR        |
| `pnpm build`   | Build for production             |
| `pnpm preview` | Preview production build locally |
| `pnpm lint`    | Run ESLint                       |

## Features

- Bilingual support (English/Spanish)
- Interactive particle starfield background
- Glassmorphism UI components
- Responsive design
- Custom CSS animations (shimmer, float, scan effects)

## Project Structure

```
src/
├── App.jsx      # Main application component with all UI
├── main.jsx     # React entry point
└── index.css    # Tailwind imports and custom animations
```
